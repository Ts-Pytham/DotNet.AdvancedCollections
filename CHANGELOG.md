## [1.0.0] - 2025-02-09

### Added
- Initial release of DotNet.AdvancedCollections
- **DoublyLinkedList&lt;T&gt;**: Bidirectional linked list with O(1) insertion/deletion
- **SortedList&lt;T&gt;**: Automatically sorted list with ascending/descending criteria
- **PriorityQueue&lt;T&gt;**: Heap-based priority queue
- **Deque&lt;T&gt;**: Double-ended queue with O(1) operations at both ends
- **PriorityStack&lt;T&gt;**: Stack with priority-based pop operations
- **BinarySearchTree&lt;T&gt;**: Binary search tree with in-order, pre-order, and post-order traversals
- **Graph&lt;TVertex, TEdge&gt;**: Directed graph with vertex and edge management
- XML documentation for all public APIs
- Full DocFX documentation website
- Comprehensive unit tests
- Support for .NET 8.0
- MIT License

### Features
- Generic implementations with strong type safety
- Nullable reference types enabled
- Modern C# language features (C# 12.0)
- Production-ready implementations
- Clean and explicit APIs
